# Grid

The grid uses a `928px` max-width. Grid columns collapse down to a single width
on small screens.


## Grid rows

A `.grid` represents a single row. For multiple rows, use multiple grids.

```html
<div class="grid">
  <div class="grid-width-4"></div>
  <div class="grid-width-8"></div>
</div>
<div class="grid">
  <div class="grid-width-8"></div>
  <div class="grid-width-4"></div>
</div>
```
